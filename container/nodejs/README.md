# Parse Backend repo

## URLs

- Parse URL: http://localhost:1337/parse
- GraphQL: http://localhost:1337/graphql
- Dashboard: http://localhost:1337/dashboard
## Local Usage

### Env Variables

Copy `.env.example` to `.env` and adjust its values according to your needs.

### Install dependencies

```
~$ npm install
```

### Run BE

```
~$ node index.js
```
